> Website and documentation of [Nuxt](https://nuxtjs.org/)

## Local Development

```bash
git clone https://github.com/nuxt/nuxtjs.org.git
cd nuxtjs.org/
yarn install
```

Start the development server on [http://localhost:4000](http://localhost:4000)

```bash
yarn dev
```
